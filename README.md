## 使用
分别cd到backEnd和frontEnd `npm install`，安装依赖包
```
npm install
```
分别cd到backEnd和frontEnd`npm run serve`，启动应用
```
npm run serve
```

## 打包
cd到frontEnd`npm run build`
```
npm run build
```
